Main Dialog Layout


This is the main dialog of the fsr program. From this dialog, it is possible to invoke both of the basic fsr operations: Conversion and Playback.

Conversion
A conversion operation requires you to specify both the input and the output file names; the conversion type is implied by the extensions of both files. If the specified conversion operation is supported, the Convert button will be enabled; pressing it will initiate the conversion operation.

Playback
A playback operation is selected by entering an input file and checking the Play in simulator radio button. The playback method is then specified in the drop-down list that is to the right of the button. If the playback operation indicated by this selection is supported, then the Play button will be enabled; pressing it will initiate the playback process.

The following set of playback methods is currently supported:

Method Description
Disk File Plays the disk file specified in the Input section directly in the simulator. If a conversion is necessary (e.g. if the simulator does not support direct playback of the file type indicated in Input, a temporary file will be created, played and then erased. The playback file's name is "handed" to the Simulator instead of sending the contents of this file to the simulator as a stream.
Pipe Plays the disk file specified in the Input section directly in the simulator, but uses an "anonymous pipe" to stream the video to the simulator instead of handing the file's name to the simulator, as in the case of "Disk File" (above). Also, as in the case of "Disk File", a temporary file is created (and then deleted) if necessary in order to carry out this operation.
FSUIPC This operation works like "Pipe" (above), but instead of using an anonymous pipe to send the video the simulator, it uses Peter Dowson's FSUIPC facility, which must be installed and configured (follow the link for more information).

Note that the Play and Convert buttons occupy the same space (in the upper right corner of the dialog); only one of them is visible at a given instant, depending upon which operation is appropriate for the set of parameters currently entered.

File Types

The set of supported file types and conversions/operations is listed in the following tables:

Type Description
FSR Microsoft Flight Simluator Video
XML FSR file in text/XML format
NMEA NMEA Data Stream
GPX GPS eXchange Format
CTL Cetus Track Log (PalmOS PDB)
PNL Pathaway NMEA Log (PalmOS PDB)
STAT Statistical Dump of Simulator Parameters
